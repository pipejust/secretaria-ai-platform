Listas Dinámicas

Para iterar listas (como Participantes, Tareas o Temas), debes crear una tabla en Word e insertar la etiqueta de inicio y fin ({{ '{' }}%p for item in array %{{ '}' }}) en la primera y última celda, y las variables de cada item en medio de ellas.

Asistentes / Participantes de la Reunión
Cíclo Inicio: {{ '{' }}%p for a in attendees %{{ '}' }} {{ '{' }}{{ '{' }} a.name {{ '}' }}{{ '}' }} {{ '{' }}{{ '{' }} a.role {{ '}' }}{{ '}' }} {{ '{' }}{{ '{' }} a.entity {{ '}' }}{{ '}' }} Cíclo Fin: {{ '{' }}%p endfor %{{ '}' }}
Temas Tratados y Puntos de Discusión
Cíclo Inicio: {{ '{' }}%p for t in themes %{{ '}' }} Título Tema: {{ '{' }}{{ '{' }} t.theme_name {{ '}' }}{{ '}' }}
Para los puntos de cada tema (Sublista):
SubCíclo Inicio: {{ '{' }}%p for d in t.discussion_points %{{ '}' }} Punto (item): {{ '{' }}{{ '{' }} d {{ '}' }}{{ '}' }} SubCíclo Fin: {{ '{' }}%p endfor %{{ '}' }}
Cíclo Fin: {{ '{' }}%p endfor %{{ '}' }}
